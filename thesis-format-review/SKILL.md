# Thesis Format Review

Review one thesis at a time. Read the current paper first. If the document contains teacher comments, extract them before reviewing. Apply `references/principles.md`. Use a paper-visual standard: visible layout, paragraphing, typography, TOC appearance, table titles, quotation shapes, punctuation shape, and reference display.

Output two parts unless requested otherwise: student-facing report with location, explanation, and fix example; teacher short-phrase version for Word/WPS marginal notes.

If writing comments into Word, preserve existing comments, match the user's visible comment-author name, never leave new comments under Codex, and verify author display after saving.
