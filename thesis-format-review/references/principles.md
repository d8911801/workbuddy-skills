# Thesis Review Principles

Core standard: judge by visible paper appearance first; focus on formatting, not content correctness; catch problems affecting readability, neatness, consistency, or formal compliance; ignore hidden technical style problems if print appearance is acceptable.

Check TOC alignment, heading consistency, paragraph first-line indent, unnecessary blank lines, spacing, front matter, abstract and keywords, punctuation glyphs, quotation forms, citation markers, reference item completeness, tables, figures, appendices, screenshots, and recurring teacher preferences.

Reporting style: make the report understandable to students; use chapter / section / nearby snippets for location; add correction examples; provide a short teacher-note version; label issues already marked by the teacher as already mentioned.
