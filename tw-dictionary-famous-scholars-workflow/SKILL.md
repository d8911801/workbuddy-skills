# 台湾现代训诂名家补充工作流

适用：台湾“现代训诂名家”条目改写/补写、与对方底稿去重比对、只提取“对方没有、我方有”的补充名家、生成可交付清单。

硬规则：先比对再写；先从对方底稿抽取台湾段已收录人名，与我方候选名单做差集；对方已有条目直接跳过；仅处理缺项。任何 docx 任务优先使用 Documents skill；Word 交付件必须做乱码检查。

条目结构：生平任职、学术职务、研究领域、学术贡献、主要论著。不确定信息标“待确认”，不得自行补造。台湾同名校写作需区分为“××大学（台湾）”。

大任务需先给规模预估与分批建议，待用户确认后执行。
