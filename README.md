# WorkBuddy Skills

Shared WorkBuddy skills for cross-agent use.

## Available Skills

### research-sync

Academic literature search and sync workflow. Place SKILL.md into ~/.workbuddy/skills/research-sync/ to install.

### academic-manuscript-collaboration-workflow

Academic manuscript collaboration workflow for Word review drafts, tracked revisions, comment replies, source-based scholarly revision, and protecting user edits. Place the folder into `~/.codex/skills/academic-manuscript-collaboration-workflow/` to install.
