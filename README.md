# WorkBuddy Skills

Shared WorkBuddy skills for cross-agent use.

## Available Skills

### academic-manuscript-collaboration-workflow

Academic manuscript collaboration workflow for Word review drafts, tracked revisions, comment replies, source-based scholarly revision, and protecting user edits. Place the folder into `~/.codex/skills/academic-manuscript-collaboration-workflow/` to install.

### integrate-student-thesis-into-draft

用於把學生畢業論文與其開題報告、問題研究表、教師批註、既有著作觀點整合為可納入專書初稿的工作流。Place the folder into `~/.codex/skills/integrate-student-thesis-into-draft/` to install.

### journal-abstract-pdf-workflow

Process yearly journal-article Excel trackers for Chinese-language research workflows. Place the folder into `~/.codex/skills/journal-abstract-pdf-workflow/` to install.

### qwen-manuscript-to-word

Prepare a fixed SOP for cloud Qwen manuscript polishing, then turn Qwen's pure-text Markdown output into independent Word drafts or tracked-revision Word files with real footnotes. Place the folder into `~/.codex/skills/qwen-manuscript-to-word/` to install.

### research-sync

Academic literature search and sync workflow. Place the folder into `~/.codex/skills/research-sync/` to install.

### thesis-defense-question-prep

Prepare live oral-defense questions for graduation theses in Word or PDF format. Place the folder into `~/.codex/skills/thesis-defense-question-prep/` to install.

### thesis-format-review

用于审查中文本科毕业论文格式，尤其是汉江师范学院及相近院校的格式要求。Place the folder into `~/.codex/skills/thesis-format-review/` to install.

### tw-dictionary-famous-scholars-workflow

用于《中国训诂学大辞典（先秦至今）》台湾“现代训诂名家”补充任务。Place the folder into `~/.codex/skills/tw-dictionary-famous-scholars-workflow/` to install.

### windows-ocr-book-workflow

Use for Windows-native OCR, split, verify, and archive work on Chinese PDF books. Place the folder into `~/.codex/skills/windows-ocr-book-workflow/` to install.

### 我的文體潤稿

用於依使用者近年親自定稿的「我的文體」進行潤稿、改寫、語氣校準與書稿口徑統一。Place the folder into `~/.codex/skills/我的文體潤稿/` to install.

### 研究史寫作

用於起草、改寫或壓縮中文學術書稿中的研究史段落，並讓研究史服務正文與結論。Place the folder into `~/.codex/skills/研究史寫作/` to install.
